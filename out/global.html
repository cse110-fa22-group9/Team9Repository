CreateActionListeners(data, seasonNumber, shadowDom)

This function takes in the current selected season (chosen by clicking a button) and then generates event listeners to switch seasons, if a season is clicked it will take update the current html to reflect the checkbox array for that season For the checkboxes which are displayed depending on the season selected, event listeners are generated so that data.episodeArray booleans are updated depending on if the checkboxes are checked or unchecked
